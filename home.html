    Automated Optimization of Pre-Production Decision Making

    Objective: Streamline the pre-production phase by automating decisions related to fabric selection, cost estimation, and necessary materials.

    Techniques: We use weighted scoring models, ridge regression, and machine learning to predict fabric durability, optimize thread combinations, and estimate costs.

    Sewing Machine Needle Performance and Stitch Monitoring System

    Objective: Enhance needle safety and performance through real-time monitoring and predictive maintenance.

    Techniques: YOLO object detection with an ESP32-CAM module for identifying needle fragments, and Hall effect sensors for needle usage monitoring with a Grafana dashboard.

    Sewing Machine Predictive Maintenance Monitoring System

    Objective: Predict and prevent sewing machine failures through data analysis and predictive modeling.

    Techniques: XGBoost for predictive maintenance, using Flask and React for data processing and user interface.

    Workforce Optimization and Operation Efficacy Monitoring System (WOEMS)

    Objective: Improve labor efficiency and operational productivity through data-driven insights.

    Techniques: Random Forest Regression for predicting labor performance, React and D3.js for dynamic data visualization, and Chart.js for interactive charts.

    Research Problem

    The apparel industry, particularly medium-scale businesses, faces significant challenges in maintaining high operational efficiency, consistent product quality, and minimizing machine downtime due to outdated manual processes. In particular, critical factors such as pre-production decision-making, workforce optimization, and sewing machine maintenance, especially needle performance monitoring, are hindered by limited technological integration, leading to increased costs, inefficiencies, and inconsistent output. The absence of automated systems for real-time monitoring and predictive maintenance of sewing machines results in frequent breakdowns, particularly due to needle wear and breakage, which disrupts production timelines and inflates maintenance costs. Additionally, manual labor allocation fails to optimize workforce productivity, with underutilization and overutilization being common. In pre-production stages, inaccurate fabric thread selection and cost estimations further compound inefficiencies.

    Research Objectives

    This research aims to revolutionize the apparel manufacturing process by integrating IoT-based solutions with machine learning to address key challenges. The primary focus is to enhance real-time monitoring of sewing machines, enabling manufacturers to track machine performance remotely and efficiently. This platform will provide continuous data streams, helping medium-scale manufacturers respond to performance issues promptly. Additionally, predictive maintenance strategies will be implemented using machine learning algorithms to anticipate machine failures. This will minimize unexpected downtime and reduce maintenance costs by shifting from reactive to proactive maintenance practices. To further streamline operations, automated systems will be developed to optimize pre-production decisions. These systems will assist in fabric selection, cost estimation, and production scheduling, thereby improving overall planning accuracy and minimizing resource wastage. The research also focuses on workforce optimization by designing a system that monitors employee performance metrics in real time. This system will enable dynamic task allocation, boost labor efficiency, and enhance operational productivity through better task management. Together, these objectives aim to increase efficiency, reduce costs, and improve the competitiveness of medium-scale apparel manufacturers.

    Methodology

    This research leverages IoT and machine learning technologies to improve various aspects of apparel manufacturing, including pre-production decision-making, machine performance monitoring, predictive maintenance, and workforce optimization. Pre-production optimization focuses on fabric and thread selection, cost estimation, and scheduling accuracy. A weighted scoring model and regression algorithms are used to predict the durability of single, dual, and tri-blend fabrics based on tensile strength tests. Additionally, the system suggests optimal fabric-thread combinations aligned with end-use requirements, budget constraints, and future cost projections to enhance efficiency and profitability before production begins. In the area of sewing machine monitoring, the study integrates needle performance tracking and predictive maintenance. The needle monitoring system employs YOLO-based machine learning models to detect and measure needle fragments after breakage, ensuring safety and minimizing risks. The Hall effect sensor system monitors needle movement in real time, alerting users to replace needles before failure occurs. Predictive maintenance extends to the sewing machines using XGBoost models trained on historical data to forecast the Remaining Useful Life (RUL) of components. A voltage monitoring system regulates power to machines, ensuring safe operations and preventing electrical malfunctions. The Workforce Optimization and Operational Efficacy Monitoring System (WOEMS) uses Random Forest Regression (RFR) to analyze employee performance and allocate tasks based on efficiency. A Grafana-based dashboard displays real-time worker metrics, while React and Chart.js enable dynamic visualization of productivity trends. Predictive analytics are employed to prevent performance declines by recommending corrective actions. This data-driven approach ensures smooth operations, enhances labor productivity, and minimizes inefficiencies through proactive decision-making.
